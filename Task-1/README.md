## Противоречия в ТЗ
- Пункты **2** и **9** - Удаление товара по кнопке в п.2 и удаление товара при уменьшении количества до 0 в п.9 (два механизма удаления + противоречие п.9 с п.1 ниже)
- Пункты **1** и **9** - п.1 написано минимум товара 1, а п.9 допускает 0 (прямое противоречие)
- Пункты **7** и **13** - п.7 цена факсированая после добавления в корзину, а п.13 цена меняется прямо в корзине (взаимоисключающие требования)
